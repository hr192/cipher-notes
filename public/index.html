About Cipher Notes

                

                    
What?

                    
A simple encrypted note sharing site. All encryption happens client-side, so the server never sees your content in plaintext.

                


                

                    
Why?

                    
Share sensitive information securely with a low barrier to entry. No accounts, no complexity‚Äîjust paste and share.

                


                

                    
How?

                    
Your browser encrypts data using AES-256 encryption. The encryption key is stored in the URL fragment (after the #), which is never sent to the server.

                


                

                    
Security

                    
    
                        
  • Zero-knowledge: Server never sees plaintext data
    • 
                        
  • AES-256: Industry-standard symmetric encryption
    • 
                        
  • URL fragments: Key never transmitted to server
    • 
                        
  • Client-side: All processing happens in your browser
    • 
                    

                


                

                    
Browser Support

                    
Requires a modern browser with Web Crypto API support (Chrome 37+, Firefox 34+, Safari 11+, Edge 79+).

                


                

                    
Privacy

                    
We log source IP, User-Agent, and creation timestamp for abuse prevention only. No analytics, no tracking, no third-party services.
